FASHION RECOMMENDATION SYSTEM

A minor project that harnesses the power of deep learning and collaborative filtering to deliver personalized, FASHION RECOMMENDATION SYSTEM outfit recommendations powered by Smart Technology.

📌 Project Overview

The FASHION RECOMMENDATION SYSTEM addresses the challenge of outfit discovery by combining image recognition with user preference modeling. Users can upload a reference outfit image which is processed by a ResNet50 convolutional neural network to extract visual feature vectors.

These features, combined with user profile attributes (occasion, gender, season), are fed into a K-Nearest Neighbors (KNN) classifier and enhanced with Collaborative Filtering to surface outfits that similar users have loved — creating a personalized, intelligent wardrobe assistant.

🎯 Project Objectives

1
Design and implement a deep learning pipeline using ResNet50 to extract meaningful visual features from fashion images.
2
Build a KNN-based recommendation engine that finds similar outfits based on extracted feature vectors and user-defined filters.
3
Implement Collaborative Filtering to leverage community preferences and improve recommendation personalization over time.
4
Develop a full-stack web application with Django, providing user authentication, outfit browsing, image upload, and a personal dashboard.
5
Deliver an accessible, responsive UI with dark mode, smooth animations, and an intuitive user experience.
6
Evaluate and benchmark recommendation accuracy, ensuring match percentages reflect genuine stylistic similarity.
