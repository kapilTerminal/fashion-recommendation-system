7-Step Pipeline

How It Works

Seven data processing steps transform the 44,424-image Myntra dataset into personalised outfit recommendations.

01

Inspect Dataset

Validate styles.csv against image folder — find missing & extra files.

02

Remove Corrupted

PIL verify() each JPG — drop corrupt & unreadable images from the CSV.

03

Remove Duplicates

Deduplicate rows by product ID to ensure clean, unique training samples.

04

Handle Missing Values

Impute or drop null colour, season, gender and usage columns.

05

Filter 26 Categories

Keep only Tshirts, Shirts, Dresses, Sarees, Jeans, Heels & 20 more article types.

06

Enrich Metadata

Map articleType → style (Casual, Formal, Ethnic…) and occasion (Office, Party, Wedding…).

07

Resize & Extract

Resize to 224×224 RGB for ResNet50 → KNN finds k nearest feature vectors.
