About

Fashion Recommendation System

A Smart Minor Project

🤖

AI-Powered Engine

Built on ResNet50 deep learning model pre-trained on ImageNet. Extracts 2048-dimensional feature vectors from fashion images and uses K-Nearest Neighbors (KNN) for similarity matching.

📊

Technology Stack

Python · Django · ResNet50 · KNN · SQLite · HTML / CSS / JavaScript

🎓

Minor Project

Developed to demonstrate the application of computer vision and machine learning in the fashion industry for personalized styling recommendations.
